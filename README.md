## 简介

根据windivert+tun2socks完成的进程加速程序，写这个主要是为了打游戏的。

## 配置文件

- 代理服务配置文件
```json
{
  "server": "127.0.0.1",
  "port": 1080,
  "method": "",
  "password": "",
  "apps": [""]
}
```
- 使用的话需要windivert的驱动文件
